> = | += | -= | \*= | /= | %=

```php

$result = $num1 + $num2;

$num1 -= $num2; //=> $num2 = $num2 - $num2  = 12-2 => 10
$num1 *= $num2; //=> $num2 = $num2 * $num2  = 12*2 => 24
$num1 /= $num2; //=> $num2 = $num2 / $num2  = 12/2 => 6
$num1 %= $num2; //=> $num2 = $num2 % $num2  = 12%2 => 0

echo "Number is " . $result;
?>
```
